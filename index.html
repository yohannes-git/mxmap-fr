Messagerie des communes françaises

C'est quoi ?

Une carte de toutes les communes françaises montrant quel hébergeur gère leur messagerie officielle — regroupées par département et région — à partir des enregistrements DNS publics.

Contexte

Souveraineté numérique : les hébergeurs américains sont soumis au CLOUD Act, qui permet aux autorités américaines d'accéder aux données stockées, quel que soit le pays d'hébergement physique. Cette carte rend visible le paysage actuel.

Comment ça marche ?

Le domaine officiel de chaque commune est analysé via les enregistrements DNS publics (MX + SPF) et classifié par type d'hébergeur.

Avertissement : les enregistrements DNS indiquent le routage du courrier et les expéditeurs autorisés, pas nécessairement le lieu de stockage des données.

Open data

Les données sont issues de l'annuaire service-public.fr (DILA) et de l'analyse DNS publique des enregistrements MX et SPF.

Ce projet est un fork de mxmap.ch, adapté pour les communes françaises — code source original sur GitHub. Code source de ce fork : yohannes-git/mxmap-fr.
